使用markdown写博文

Date 一 30 三月 2015 By Charles Category markdown. Tags Github Pelican markdown

1、基本格式

在content/articles目录下新建一个2015-03-30.md文件，写入以下内容：

Title:使用markdown写博文
Author:Charles
Date:2015-03-30
Category:markdown
Tags:Github, Pelican,  markdown
Slug:2015-03-30         （之后将会生成2015-03-30.html文件，如果为空则生成得html文件的文件名为标题的拼音）
Summary:本博客为使用Pelican在Github搭建，搭建方法不难，请自行搜索。本文主要记录使用markdown写博文的方法。

（此处为正文）

然后在content目录的父目录下（我的是在Blog目录下）执行：

make html

浏览器打开http://localhost:8000查看效果

2、插入图片

格式为：

1、插入本地图片
 ![显示名](../images/2015-03-30_1.png )
2、插入网络图片
![显示名](http://.../2015-03-30_1.png )

3、插入源代码

在代码的开头和结尾处使用三个反点号，格式为：

```C
#include<stdio.h>

int main()
{
    int  num;

    printf("Please enter a int number:");
    scanf("%d", &num);
    printf("You entered :%d \n", num);

    return 0;
}
```

效果为：

4、插入表格

格式：

|年份             |价格            |
|:----------------|:---------------|
|2015            |500             |
|2014            |550             |
|2015            |800             |

效果：

年份 价格
2015 500
2014 550
2015 800
